🔒 **Password Strength Checker** [ Beginner Project ]

Welcome to the **Password Strength Checker**! This is a simple web application that checks the strength of your password based on various criteria like length, complexity, and special characters. The app uses Flask on the backend and provides a sleek, neon-themed frontend to help you evaluate your password strength.

----------------------------------------------
🚀 Features:
----------------------------------------------
- 🧑‍💻 **Frontend**: A stylish user interface with a password strength bar and criteria checks.
- 🔑 **Backend**: Powered by **Flask**, the app validates password strength, hashes passwords using **SHA256**, and encrypts them with **AES**.
- 📊 **Strength Indicator**: Displays a real-time progress bar that changes color based on password strength.
- 🔒 **Security**: Passwords are checked for length, complexity, and common patterns.
-

## ⚙️ Installation
----------------------------------------------
To run this project locally, follow these steps:

1. Clone the repository:
     git clone https://github.com/YourUsername/PasswordStrengthChecker.git
   
2. Install dependencies:
     pip install -r requirements.txt
    
3. Run the application:
     python app.py

4. Open the app in your browser:
     Navigate to http://127.0.0.1:5000/ to view the password strength checker in action.

🛠️ Features & Functionality:
----------------------------------------------

🔒 Check password length (Minimum 12 characters).

🔡 Ensure at least one uppercase, lowercase, number, and special character.

💣 Detect common passwords like password, 123456, and qwerty.

🟢 Password strength progress bar that turns from red to green as the password becomes stronger.
